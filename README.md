# get_set

 Provides standard getter-setter definitions in C++

## How to use

## in cmake

 add_subdirectory(<this_directory>)
 target_link_library(your_target \<option\> get_set)

## in CXX code

 #include "get_set.hpp"

## I recommend that you precompile get_set.hpp
